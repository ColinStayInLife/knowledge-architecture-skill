# Knowledge Architecture Skill

一套完整的个人知识管理架构，包含 Obsidian 配置、AI 助手联动、记忆系统。

## 功能

- 🏠 **知识库架构** - PARA + Johnny Decimal 目录结构
- 📝 **模板系统** - 日报、项目、会议、研报模板
- 🤖 **AI 联动** - 与 OpenClaw/Clawdbot 无缝集成
- 📅 **时间流视图** - Memos 风格的时间线聚合
- 🏷️ **标签导航** - 按标签快速浏览笔记
- 💾 **记忆系统** - AI 长期记忆 + 工作日志自动整理

## 安装

```bash
npx clawhub@latest install knowledge-architecture
```

或手动安装：

```bash
git clone https://github.com/YOUR_USERNAME/knowledge-architecture-skill.git
cd knowledge-architecture-skill
./install.sh
```

## 使用

安装后，在 OpenClaw 中说：

- "帮我初始化知识库" - 创建完整目录结构
- "新建日报" - 基于模板创建今日日报
- "整理今天的工作" - 自动整理对话到日报
- "更新项目进展" - 更新项目概览
